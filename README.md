# SpaceTrigger

SpaceTrigger is a simple Python automation tool that triggers 3 Spacebar presses when the ENTER key is pressed. It can be used for timing-based applications like cstimer or other browser tools.

---

##  Features
- Detects ENTER key press
- Sends 3 Spacebar inputs
- Custom timing (short → 1s → short)
- Custom shortcats
- Setup own Times
- Works on Windows
