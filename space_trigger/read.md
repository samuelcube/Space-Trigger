# SpaceTrigger

Automatisiert 3 Spacebar-Klicks nach einem ENTER-Druck.

Für jede Webseite in jedem Browser
z.b CsTimer

Zeiten können beliebig geändert werden und entfernt werden zum beispiel für 1 klick

## Installation

Python installieren:
https://python.org

Abhängigkeiten installieren:

```bash
pip install -r requirements.txt
```
